### 1. Clone & Initialize

```bash
git clone https://github.com/yourname/mpquic-mobility.git
cd mpquic-mobility

```

### 2. Build (CMake)

```bash
mkdir build && cd build
cmake ..
make -j4
```

### 3. Run Server

```bash
./server_recv -p 4433 \
  --cert ../certs/cert.pem \
  --key ../certs/key.pem
```

```bash
./client_uploader \
   <server_ip> \
   <client_ip> \ 
   <client_mp_ip> \
```
